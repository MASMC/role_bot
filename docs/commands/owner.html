Command Syntax Description
Admin Role adminRole <Role ID|Role Mention> Sets the admin role for the server.
Create Embed createEmbed (message) (colour code in hex format, without 0x) Creates an embed using the message and colour given
Populate Roles populateRoles Populates the list of roles for the server.
Shutdown shutdown Shuts down the bot.
Staff Role staffRole <Role ID|Role Mention> Sets the staff role for the server.
Throw Error throwError (error code) Throws an error, with optional error code.
View Configs viewConfigs Outputs the config file for the server.

Also includes the "staff" and "general" commands
