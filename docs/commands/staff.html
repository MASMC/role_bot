Command Syntax Description
Add Role addRole <user mention> <role mention> Adds a role to the specified user.
Help help (general|staff|owner) Gets the link to the help page. The parameter is optional.
Remove Role removeRole <user mention> <role mention> Removes a role from the specified user.
Say say <message> Makes the bot "say" something.
View Roles viewRoles Outputs the role list for the server.

Also includes "general" commands.
